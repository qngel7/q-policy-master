## 편집 방법

- 아래 `메일 제목`부터 `하단 문구`까지 원하는 표현으로 직접 수정한다.
- `{이름}`, `{기준_Q메일}`처럼 중괄호로 표시한 부분은 회원마다 자동으로 바뀌는 값이므로 삭제하지 않는 것을 권장한다.
- 문장을 삭제하거나 순서를 바꿔도 된다.
- 버튼 이름도 자유롭게 수정할 수 있다.
- 수정이 끝나면 문서 상단의 `상태`를 `문구 확정`으로 바꾸거나, 변경 완료 사실을 작업자에게 알린다.

# 편집할 메일 원문

## 보낸 사람

Q. `<Q@Q.co.kr>`

## 메일 제목

```text
[Q메일 발급 완료] {이름}님의 새로운 이름이 시작되었습니다
```

## 미리보기 문구

```text
이 메일이 도착했다면 Q메일 발급과 기존 메일함 연결이 정상적으로 완료된 것입니다.
```

### 메일은 그대로,
### 이름은 새롭게.

{이름}님, Q메일을 통하여 Q.플랫폼의 일원이 되신것을 환영합니다.

**Gmail·네이버메일은 메일 서비스,  
Q메일은 이름 서비스입니다.**

Q메일은 지금 사용하는 네이버·Gmail·다음·Outlook·Apple Mail은 그대로 사용하면서, 다른 사람에게 알리기 쉽고 기억하기 쉬운 나만의 이름을 만드는 서비스입니다.

✅ 이 메일은 새 Q메일 주소로 발송되어 가입할 때 연결한 기존 메일함으로 전달되었습니다.

이 메일을 받으셨다면 Q메일 발급과 포워딩이 모두 정상적으로 완료된 것입니다.

## 보내는 이름 설정

### Q메일을 ‘보내는 이름’으로 사용해 보세요

현재 연결된 메일서비스는 **{연결_메일서비스}**입니다.

지금 사용하는 메일서비스는 그대로 두고, 상대방에게 보이는 이름을 다음처럼 설정할 수 있습니다.

**{추천_보내는_이름}**

이 설정은 실제 발신 이메일 주소를 바꾸는 것이 아닙니다. 기존 메일주소로 그대로 보내면서 상대방에게 보이는 이름에 Q메일을 표시하는 방법입니다.

가능한 메일서비스에서는 `답장받을 주소`도 Q메일로 설정할 수 있습니다. 그러면 상대방의 답장이 Q메일을 거쳐 현재 메일함으로 도착합니다.

### 연결 서비스에 따라 자동으로 바뀌는 짧은 안내

#### 네이버메일

1. 환경설정
2. 기본 설정 → 쓰기
3. 보내는 이름 → 직접 입력
4. 추천 이름을 입력하고 저장

선택사항: `답장받을 메일 주소`를 Q메일로 지정합니다.

**[네이버 설정 방법 자세히 보기]({서비스별_안내_URL})**

#### Gmail

1. PC에서 Gmail 열기
2. 설정 → 모든 설정 보기
3. 계정 및 가져오기 → 다른 주소에서 메일 보내기
4. 정보 수정에서 추천 이름을 입력하고 저장

선택사항: `다른 답장 주소 지정`에 Q메일을 입력합니다. Gmail 앱에서는 보내는 이름을 변경할 수 없으므로 PC 웹에서 설정합니다.

**[Gmail 설정 방법 자세히 보기]({서비스별_안내_URL})**

#### 다음·한메일

1. 설정
2. 계정관리 → 보내는 사람
3. 사용할 메일주소 선택
4. 추천 이름을 입력하고 저장

**[다음·한메일 설정 방법 자세히 보기]({서비스별_안내_URL})**

#### Outlook

Outlook 종류에 따라 설정 방법이 다릅니다. Outlook.com, 클래식 Windows Outlook, Mac용 Outlook 가운데 사용하는 환경을 안내 화면에서 선택합니다.

**[Outlook 설정 방법 자세히 보기]({서비스별_안내_URL})**

#### Apple Mail

Apple Mail은 Mac Mail과 iCloud Mail의 설정 방법이 다릅니다. 사용하는 기기와 계정을 안내 화면에서 선택합니다.

**[Apple Mail 설정 방법 자세히 보기]({서비스별_안내_URL})**

## Q 서비스 소개

### 이 이름에서 더 많은 것이 시작됩니다

**대시보드 (향후 구축)**  
나의 Q-ID·Q메일·프로필과 Q 활동 현황을 한곳에서 확인하고 관리하는 공간입니다. 앞으로 미션 참여와 Q-vic 획득·사용 내역도 대시보드에서 확인할 수 있도록 구축할 예정입니다.

**Q-vic**  
Q 활동과 혜택에 사용하는 서비스 크레딧입니다. Q에서 제공하는 미션을 수행하면 Q-vic을 획득할 수 있으며, 미션별 획득 기준과 이용 방법은 대시보드 구축 후 안내할 예정입니다.

**Q-bit NFT**  
Founder Stage 이후 단계적으로 제공될 예정인 Q 생태계의 디지털 자산입니다. 세부 혜택과 이용 시점은 멤버십에 따라 달라질 수 있습니다.

**[Q 홈 열기]({Q홈_URL})**

## 실제 발송 반영 전 확인

- 편집 원문과 HTML메일 문구가 같은지 확인한다.
- 일반 텍스트메일과 HTML메일에 같은 핵심 내용이 들어가는지 확인한다.
- 010Q에는 간편 주소 설명이 나타나지 않는지 확인한다.
- OK-Q·Founder에는 간편 주소 설명이 나타나는지 확인한다.
- 연결 메일서비스에 맞는 안내만 우선 표시되는지 확인한다.
- 다른 메일서비스 안내도 항상 열 수 있는지 확인한다.
- 시험메일 1건을 새 Q메일 주소로 보내 모바일과 PC에서 확인한다.
