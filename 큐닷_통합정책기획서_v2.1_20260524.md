# 2. Q-ID 시스템 정책

Q-ID는 큐닷 플랫폼의 **핵심 디지털 신원 인프라**. 모든 회원·AI 에이전트·전문가의 고유 식별자이자 영구 불변의 여권.

## 2-1. 기본 Q-ID 구조

| 구분 | 형식 | 순번 구간 | 비고 |
|------|------|---------|------|
| **창립회원** | 이니셜3자 + 8자리 순번 | 0000-0000 ~ 0499-9999 | 영구 우대 |
| **외국인 거주자** | 국가코드2자.이름 | 0500-0000 ~ 0999-9999 | KR 생략 |
| **일반회원** | 이니셜3자 + 8자리 순번 | 2000-0000 ~ | 한국인 KR 생략 |

### 예시
- **SDW-0000-0001** (서대원, 코어회원)
- **KMS-0500-1234** (한인 거주자, 캐나다)
- **PTS-2000-5678** (일반회원)

## 2-2. V-ID (Verified ID) 6종 — 신원 검증 체계

| V-ID 종류 | 코드 | 인증 방식 | 특징 |
|----------|------|---------|------|
| **V-ID-010** | 핵심 | 휴대폰번호 본인인증 | 비공개·2FA 생체인증·SMS 비용 무료 |
| **V-ID-Pro** | LAW/TAX/DOC 등 | 자격증 서류 인증 | 7000번대 부여·전문직 코드+순번 |
| **V-ID-Fan** | 팬덤명 | 소속사 라이선스 | BTS·BLACKPINK 등 팬덤명·글자수 무제한 |
| **V-ID-Gold** | 경매 | 경매 낙찰 | 777-7777-7777 등 희소 번호 경매 |
| **V-ID-Nic** | 닉네임 | 선착순 | 1~5자 닉네임 Q-ID |
| **V-ID-Free** | 자유 | 선착순 | 유보어 제외·무작위 서브도메인 구입자 |

## 2-3. A-ID (AI 에이전트 ID)

- **형식:** A00~A99 + 기본 Q-ID 순번 (인당 최대 100개)
- **대상:** AI 에이전트 전용 신원
- **엔드포인트:** `이름.q.co.kr/api/identity` / `llms.txt` 지원
- **B2B 에이전트 서브도메인:** `법무봇.q.co.kr` 등 (1만 회원 이후 상용화)
- **KAR (Korea Agent Registry):** 장기 비전 — AI 에이전트 신원 등록 시스템

### A-ID 생성 및 거래

- 사용자가 자신의 Q-ID 기반으로 AI 에이전트 100개까지 생성 가능
- 각 에이전트는 독립적인 신원(A00~A99)과 신뢰도 점수 보유
- 에이전트 간 Q&A 작업 거래 및 마이크로 결제 가능
- 블록체인 기반 신원 증명 (GMMT/Polygon/Kaia 등 검토 중)

# 3. 010-Q메일 시스템

**핵심 명제:** 번호가 곧 이메일, 이메일이 곧 번호.

## 3-1. 010-Q메일 표기 규칙

| 요소 | 표기 방식 | 이유 |
|------|---------|------|
| **기본 형식** | 01070008000@**Q**.co.kr (Q는 오렌지색 강조) | 하이픈 제거·점(.) 생략·Q 시각적 강조 |
| **구독회원** | [010] 힌트로 얹거나 표기 없앰, 선택적 표시·가독성 우선 | — |
| **라벨 표기** | M&E | 모바일 & 이메일 통합 개념 단축 |

**예시:** 010-7000-8000 → **01070008000@Q.co.kr** (Q 오렌지색)

**구독회원 표기 유연성:**
- 01070008000, 010-7000-8000, 010.7000.8000, 7000-8000, 7000.8000, 70008000
- 위 표기 모두 가능하며 **70008000@Q.co.kr** 권장
- 메일 발신자가 위 어떤 형태로 메일 발송해도 수신됨

## 3-2. 이메일 체계 전체

## 4-3. 상표권 선점 프레임 (§11)

**원칙:** 상표권과 관계없이 큐닷 서브도메인은 **선착순 사용권** 부여.

### 사용권 분리 체계

| 사용권 | 귀속 | 설명 |
|-------|------|------|
| **1차 사용권** | 피분양자 (양도전까지) | 최초 피분양자가 행사·관리·양도 불가 |
| **2차 사용권** | 상표권자 (우선권) | 1차 종료 후 상표권자가 우선 취득 |

### 운영 규칙

1. **최초 피분양자:** 1차 + 2차 사용권 모두 보유 및 행사
2. **양도 시:** 1차 사용권만 양도 가능, 2차 사용권은 큐닷에 귀속
3. **상표권 분쟁:** 판결 결과에 절대 준종
4. **사용권 만료 후:** 상표권자가 2차 사용권 우선 행사권 보유

## 4-3-1. 상표권 분쟁 시 처리 절차 (예시: samsung.Q.co.kr)

# 5. 다이어트 명함 앱

**비전:** "당신의 명함이 가벼워집니다" — 기존의 복잡한 명함에서 **번호 하나 + QR코드**로 모든 것을 통합.

**QR코드 기능:**
- **Q-ID로 연결:** 명함의 QR코드 스캔 → Q-ID 프로필 페이지 자동 연결
- **컨택 지속성 보장:** 휴대폰 번호 변경 시에도 QR → Q-ID로 컨택 유지 (변경 불가침 채널)
- **선택적 표기:** 사용자가 명함 디자인 시 QR코드 노출 여부 선택 가능
- **010Q메일 병기:** QR코드 옆에 010Q메일 표기 (텍스트 복사·스캔 선택지 제공)

## 5-1. 핵심 컨셉 — 비포&애프터

| BEFORE (기존 명함) | AFTER (다이어트 명함) |
|------------------|-------------------|
| 회사명 / 직함 / 이름 / 주소 / TEL / FAX / Mobile / E-mail — **8줄** | 회사명 / 직함 / 이름 / **[QR코드] [010] + Q메일 — 4줄, 연락처 단 1줄** |

**QR코드 표기:** 선택적 (사용자 디자인 단계에서 ON/OFF 선택) — Q-ID 프로필 링크

**제작 방식:** 기존 명함 사진 위의 텍스트 영역만 제거 → Q메일 + QR코드(선택적) 오버레이 방식 (원본 질감 최대 보존) → 자동 Q-ID QR 생성

## 5-2. 앱 UX 플로우

| 단계 | 액션 | 내용 |
|------|------|------|
| **①** | **스캔** | 스마트폰 카메라 촬영 → AI OCR 자동 추출 (한국어·한자·영어·일본어 동시 인식) |
| **②** | **변환** | 원본 명함 위 주소·TEL·FAX·Mobile·E-mail 제거 → [010] + Q메일 오버레이 → 비포&애프터 렌더링 |
| **③** | **전송** | 카카오톡 / 문자로 변환 명함 즉시 공유 — 상대방 WOW 경험 유도 |

| **3단계** | 큐빗·큐빅 모두 0 | 계정 동결 (재활동 신청 필요) |

**예시:** 창립회원이 3개월 미활동 → 큐빅 ₩150,000 → ₩147,000 → ₩144,000 → 0 → 큐빗 100 → 99 → 98...

## 10-2. 큐빗(Cubit) 경제 — Web3 변동 자산 (Crypto) 하이브리드 전략

**핵심 원칙:** 규제 리스크 최소화 = 플랫폼 내부 결제는 큐빅(스테이블)으로 격리 + 외부 자본 유입은 큐빗(변동)으로 연결

| 단계 | 통화 | 역할 | 규모 |
|------|------|------|------|
| **MVP~10만 명** | 큐빅 | 플랫폼 내부 통화 (스테이블) | Supabase DB |
| **10만~100만 명** | 큐빅 + 큐빗 | 포인트 + 온체인 | Supabase + 스왑 브릿지 |
| **100만명+** | 큐빗 | 주요 거래소 상장 (변동) | 메인넷 (Polygon/Kaia/ICP 검토) |

### 환율 정책

**큐빅(Cubic) 국가별 가치 체계:**

| 국가 | 코드 | 최소단위 | 기준 가치 | 예시 |
|------|------|---------|---------|------|
| **한국** | KR | 1원(₩) | 가장 인기있는 식빵 1개 (약 ₩3,000) | 1큐빅 = 1원 |
| **일본** | JP | 1엔(¥) | 가장 인기있는 식빵 1개 (약 ¥100) | 1큐빅 = 1엔 |
| **미국** | US | 1센트($) | 가장 인기있는 식빵 1개 (약 $3) | 1큐빅 = $0.01 |
| **유로존** | EU | 1센트(€) | 가장 인기있는 식빵 1개 (약 €1) | 1큐빅 = €0.01 |

**근본 가치 기준:** 각 국가에서 가장 인기있는 빵(식빵/롤빵) 1개의 현지화폐 가격을 기준으로 정기 재평가 (분기 1회)

**큐빗(Cubit) 통일 기준:**
- **1 큐빗 = 변동 가치:** 초기 1달러(USD) 페깅 (거래소 상장 후 변동)
- **스왑 브릿지:** 조건 충족 유저(예: 레벨 3+, 생체 인증) → 큐빅 ↔ 큐빗 양방향 교환 허용

# 11. 인증 및 SSO 전략

## 11-1. 인증 vs 신원 (Two-Layer)

큐닷 생태계에서 사용자는 **'인증(Authentication)'과 '신원(Identity)'** 두 가지 층위로 관리됨.

| 계층 | 기술 | 역할 | 비유 |
|------|------|------|------|
| **인증 레이어** | Wepin + Google + 생체인증 | 당신이 "구글 계정 소유자"임을 증명 | 현관문을 여는 '열쇠' |
| **신원 레이어** | Q-ID | 큐닷 플랫폼 내 영구 불변 고유 번호 | 제국 어디서든 통용되는 '여권' |

## 11-2. SSO (Single Sign-On) 작동 원리

**목표:** 메인 포털 한 번의 로그인 → 모든 서브도메인 자유롭게 이용

| 단계 | 프로세스 |
|------|---------|
| **1. 인증 전파** | q.co.kr에서 Wepin+Google 로그인 → 세션 정보 생성 |
| **2. 자동 인식** | 유저가 shop.q.co.kr / news.q.co.kr 접속 → 백엔드에서 세션 자동 확인 |
| **3. 추가 로그인 배제** | 입점사 사이트에서 별도 아이디/비밀번호 입력 불필요 → 이탈률 극적 감소 |

## 11-3. 사용자 경험 (UX) 시나리오

1. **접속:** 사용자가 q.co.kr (또는 큐.한국 / 큐닷.한국) 메인 페이지 → "구글 로그인" 선택
2. **인증:** Wepin SDK가 구글 계정 검증 → 디지털 지갑 세션 오픈
3. **매핑:** 시스템이 검증된 구글 계정 ↔ Q-ID (예: SDW-0000-0001) 매핑 확인
4. **활동:** 사용자는 쇼핑몰, 뉴스, Q&A 등 모든 서브도메인을 '지주(SDW-0000-0001)' 자격으로 이용

## 11-4. 기대 효과

| 효과 | 설명 |
|------|------|
| **데이터 통합** | 모든 활동 데이터가 하나의 Q-ID에 축적 → 초정밀 유저 타겟팅 가능 |
| **비용 절감** | 입점사들은 별도 보안/인증 서버 구축 불필요 + SMS 인증 비용 부담 사라짐 |
| **보안 강화** | 생체 인증(Passkey) + Wepin 지갑 보안 결합 → 개인정보 유출 리스크 최소화 |

# 14. 핵심 원칙 및 한류 포지셔닝

## 14-1. 비즈니스 철학: 네트워크가 자산

**근본 원칙:**

1. **자본이 아니라 네트워크가 자산**
   - 가입자가 먼저 쌓이고 → 가입자가 협상력 → 협상력이 사업권이 된다
   - 거대 자본 없이 회원 베이스로 스타링크 법인 계약 협상 가능

2. **수익 구조 우선순위**
   - 1순위: 서브도메인 임대 (디지털 부동산)
   - 2순위: 명함 주문 (오프라인 락인)
   - 3순위: CPL/매칭 (전문가 연결)
   - 4순위: 광고 CPM (트래픽 기반)
   - 5순위: 데이터 B2B (진성 데이터)
   - 6순위: 구독료 (선택)

3. **제약을 기술로 극복**
   - 비개발자 제약 → 노코드/로코드 + API 조합
   - 무투자 제약 → 선납금 + 협동조합 모델
   - 인프라 제약 → Cloudflare + Supabase 무료 스택

4. **Virtuous Cycle 창조**
   - 명함 한 장 → Q메일 → MVNO → 위성통신 → 글로벌 한류 플랫폼

## 14-2. 한류 구조 속 Q.의 포지션

> BTS는 한류의 **얼굴**, Q.co.kr은 한류의 **주소**가 된다

| 주체 | 역할 | 팬과의 관계 | 지속성 |
|------|------|----------|-------|
| **BTS·K-아티스트** | 콘텐츠·아티스트 | 바라보는 대상 | 활동 기간 有 |
| **삼성·현대** | 제품·기술 | 구매하는 대상 | 국적 희석 |
| **Q.co.kr** | 플랫폼·정체성 | **소속되는 장소** | **한계 없음** |

## 14-3. Hallyu Passport — 한류 여권

**비전:** Q-ID = K-멤버십 배지 — co.kr 주소를 가진 전 세계 K-팬
