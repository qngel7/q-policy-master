## 신설/개정 조문

- **§7-0 회원아이디 발급 규칙 (신설)** — 기본 Q-ID·V-ID 형식, 유보 구간 정의, 발급 시점(temp→active 트리거).
- **§7-1** — 번호 구간 행을 회원아이디 규칙으로 교체, 혜택표에 'V-ID 추가 발급' 행 추가.
- **§7-2** — 등급 매트릭스의 '구간' 컬럼을 '기본 회원아이디(공통순번)/추가 ID'로 재구성.

## 시스템 반영 (services/010)

- `db/trigger_assign_qid.sql` — 단일순번 기본 ID + 창립 V-ID 자동발급 (idempotent: q_id가 TEMP일 때만).
- `db/add_v_id_column.sql` — `v_id` 컬럼 + UNIQUE.
- `db/reserved_vid.sql` — `reserved_vid_numbers` 테이블 + `is_vid_selectable()` (일반 V-ID 발급은 보류, 구조만).
- `db/migrate_founder0.sql` — 기존 #0(서경석) 표준 정규화 (V-ID `SKS00000000` + 공통순번 기본 ID).
- `db/schema.sql`, `index.html`, `functions/api/lookup.js` — 형식 표시·v_id 반환 반영.
