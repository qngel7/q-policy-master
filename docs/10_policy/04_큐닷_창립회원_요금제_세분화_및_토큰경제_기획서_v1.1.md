## §4. Q 이니셜 메일 체계 (신설)

### 4-0. 이름메일 2슬롯 원칙 (자원 배분 정책)

이름메일 주소는 동명이인이 존재하는 **한정 자원**이다.  
1인이 보유할 수 있는 이름 기반 메일은 **슬롯 A + 슬롯 B = 최대 2개**로 제한한다.

| 슬롯 | 종류 | 내용 | 비고 |
| :---: | :--- | :--- | :--- |
| **A (고정)** | 한글이름@Q | `서대원@Q.co.kr` | OK-Q 이상 전원, 변경 불가 |
| **B (택 1)** | 영문여권이름@Q | `WEDAEHAN@Q.co.kr` | Founder 즉시 발급 기본값 |
| **B (택 1)** | 이니셜 메일 | `Seo.DW@Q.co.kr` 등 | 이니셜 선택 시 슬롯 B 점유 |

**핵심 규칙: 슬롯 B는 하나만**

```
이니셜 메일 선택  →  영문여권이름@Q 발급 안 됨
영문여권이름@Q 보유  →  이니셜 메일 추가 불가
```

> **변경:** 슬롯 B를 한 번 확정한 뒤 변경하려면 별도 정책 절차 필요 (향후 정의).  
> **010-Q:** 이름메일 슬롯 없음 (번호 기반 메일만).  
> **OK-Q:** 슬롯 A 예약 + 슬롯 B(영문여권이름) 예약 = 2개 예약권.

---

### 4-1. 정의

**Q 이니셜 메일**이란 Founder 플랜 회원이 **원할 경우 추가로 신청하는** 선택형 브랜드 메일이다.  
영문여권이름@Q가 Founder 기본 발급 메일이고, 이니셜 메일은 그 위에 얹는 추가 옵션이다.

| 구분 | 슬롯 | 형태 | 발급 방식 |
| :--- | :---: | :--- | :--- |
| OK-Q 한글이름 | A | `서대원@Q.co.kr` | **예약권** |
| OK-Q 영문여권이름 | B | `WEDAEHAN@Q.co.kr` | **예약권** |
| Founder 영문여권이름 | B (기본값) | `WEDAEHAN@Q.co.kr` | **즉시 발급** |
| Founder 이니셜 (선택) | B (대체) | `Seo.DW@Q.co.kr` 등 | **즉시 발급** (영문여권이름 대신) |

> **슬롯 B는 하나만:** 이니셜 메일을 선택하면 영문여권이름@Q는 발급되지 않는다.  
> OK-Q → Founder 업그레이드 시: 슬롯 B 예약(영문여권이름)이 즉시 발급으로 전환, 또는 이니셜로 교체 선택.

### 4-2. Standard / Starter 이니셜 메일 형식

**Standard Founder 기본 형식 (`Seo.DW@Q.co.kr`)**

```
[성 영문].[(이름 각 음절 로마자 첫 글자 대문자)]@Q.co.kr
```

| 한글 이름 | 성 영문 | 이름 이니셜 추출 | 이니셜 메일 |
| :--- | :--- | :--- | :--- |
| 서**대**원 | Seo | **D**ae + **W**on → DW | `Seo.DW@Q.co.kr` |
| 김**민**준 | Kim | **M**in + **J**un → MJ | `Kim.MJ@Q.co.kr` |
| 이**수**연 | Lee | **S**u + **Y**eon → SY | `Lee.SY@Q.co.kr` |
| 박**지**훈 | Park | **J**i + **H**un → JH | `Park.JH@Q.co.kr` |
| 최**예**린 | Choi | **Y**e + **R**in → YR | `Choi.YR@Q.co.kr` |
| 정**재**우 | Jung | **J**ae + **W**oo → JW | `Jung.JW@Q.co.kr` |

**이름 이니셜 추출 규칙:**
- 이름 각 음절의 **로마자 표기 첫 알파벳** 대문자 사용
- 음절 수 = 이니셜 글자 수 (2음절 이름 → 2자, 3음절 이름 → 3자)
- 성씨 영문: 국립국어원 표준 로마자 표기법 기준
  - 서=Seo, 김=Kim, 이=Lee, 박=Park, 최=Choi, 정=Jung, 강=Kang, 윤=Yoon, 장=Jang, 임=Lim

#### Starter `SDW@Q.co.kr` 충돌 시

| 순위 | 대안 형식 | 예시 |
| :---: | :--- | :--- |
| 1 | **선점** | `SDW@Q.co.kr` (선점자 유지) |

### 4-4. VIP / Genesis 메일 체계

**VIP `DW@Q.co.kr` (이름 이니셜만, 숏메일)**

- 성씨 없이 이름 이니셜 2~3자만 사용 → 더 짧고 희소
- 기존 `Seo.DW@Q.co.kr`도 동시 활성화 (포워딩)

**Genesis 커스텀 (2~5자 자유 선택)**

- `Won@Q.co.kr`, `Seo@Q.co.kr`, `DaeWon@Q.co.kr` 등 본인이 영문 직접 지정
- 단, **이미 등록된 사용자명과 중복 불가**, **시스템 예약어 불가**
- 선택 불가 시 VIP 숏메일 규칙으로 폴백

**전체 메일 등급 요약:**

```
[선택] Genesis:   Won@Q.co.kr          ← 커스텀 2~5자 (최희소)
[선택] VIP:       DW@Q.co.kr           ← 이름이니셜만 (숏)
[선택] Standard:  Seo.DW@Q.co.kr       ← 성.이름이니셜 (이름형)
[선택] Starter:   SDW@Q.co.kr          ← 성+이름 이니셜 3자 연속
─────────────────────────────────────────────
[기본] Founder:   WEDAEHAN@Q.co.kr     ← 영문여권이름 즉시 발급
[예약] OK-Q:      서대원@Q.co.kr 외     ← 한글+영문 이름메일 예약권
[번호] OK-Q:      12345678@Q.co.kr     ← 8자리 번호
[번호] 010-Q:     01012345678@Q.co.kr  ← 11자리 번호
```

### 4-5. 발급 프로세스

**A. 영문여권이름@Q (Founder 기본, 자동)**

1. Founder 가입 완료 시 — 영문이름 입력값 기준 `WEDAEHAN@Q.co.kr` 자동 생성
2. 중복 없으면 즉시 활성화, 중복이면 숫자 차순(`WEDAEHAN2@`) 또는 사용자 선택
3. 가입 완료 화면에 발급된 메일 주소 표시

**B. 이니셜 메일 (선택, 별도 신청 — 슬롯 B 대체)**

1. 가입 완료 후 마이페이지에서 "이니셜 메일 신청" 선택
2. **⚠️ 안내 표시:** "이니셜 메일을 선택하면 영문여권이름@Q는 발급되지 않습니다"
3. 사용자 확인 → 티어별 허용 형식 자동 표시 → 이니셜 자동 추출 후 가용 여부 확인
4. 가용 시 즉시 발급 (슬롯 B 확정), 충돌 시 §4-3 Cascade 순서로 대안 제시
5. 확정 후 슬롯 B 잠금 — 영문여권이름@Q 발급 요청 시 "슬롯 B 이미 사용됨" 안내

## §6. 자연어·브랜드 서브도메인 & 메일 세트 (자유 구매)

이름 기반 메일과 별개로, **자연어 단어·브랜드명·직업·관심사** 기반의 주소는  
개수 제한 없이 자유롭게 구매할 수 있다. 큐닷의 "디지털 부동산" 상품군이다.

### 6-1. 상품 구성

| 상품 | 내용 | 예시 |
| :--- | :--- | :--- |
| **메일 단독** | 원하는 영문 단어@Q.co.kr | `Doctor@Q.co.kr`, `Coffee@Q.co.kr` |
| **서브도메인 단독** | [단어].q.co.kr 영구 사용권 | `doctor.q.co.kr`, `seoul.q.co.kr` |
| **메일 + 서브도메인 세트** | 메일과 서브도메인 동일 키워드 묶음 | `Doctor@Q.co.kr` + `doctor.q.co.kr` |

### 6-2. 이름메일과의 차이

| 구분 | 이름메일 (§4) | 자연어·브랜드 세트 (§6) |
| :--- | :--- | :--- |
| **성격** | 신원(Identity) — 개인 아이덴티티 보호 | 자산(Asset) — 디지털 부동산 투자 |
| **한도** | 2슬롯 (A+B) 엄격 제한 | **무제한** 구매 가능 |
| **가격** | 플랜 가입비 포함 | 별도 유료 (단어 희소성에 따라 가격 차등) |
| **예시** | `서대원@Q.co.kr`, `Seo.DW@Q.co.kr` | `Lawyer@Q.co.kr`, `Seoul.q.co.kr` |
| **목적** | 내가 누구인지 | 내가 무엇을 하는지 / 무엇을 소유했는지 |

### 6-3. 가격 책정 원칙 (안)

- **일반 단어** (5자 이상): 기본가 (연간 또는 영구 일시불)
- **희소 단어** (4자 이하, 국가명, 직업명 등): 프리미엄 가격
- **1~2자 슈퍼숏** (`AI@Q.co.kr`, `Go@Q.co.kr` 등): 경매 또는 별도 정책
- **결제 수단**: 현금 또는 Q-vic (Q-vic 결제 시 50% 캐시룰 적용)

> 자연어 주소의 구체적인 가격표 및 경매 시스템은 별도 기획서에서 정의.
