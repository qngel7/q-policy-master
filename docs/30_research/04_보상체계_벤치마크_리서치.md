## ★ Executive Summary — Q-vic 권장 단가

리서치 결과를 종합한 **Q-vic 보상 권장 단가** (단위: Q-vic = 원):

| 활동 | 시장 평균 | **Q. 권장 단가** | 일일 상한 | 월 최대 |
|---|---|---|---|---|
| **출석체크 (단순)** | ₩10~100/일 | **30 Q-vic/일** | 30 | ₩900 |
| **만보기 (걷기)** | ₩50~100/일 | **20 Q-vic/일 (선택)** | 20 | ₩600 |
| **검색 + 체류** | ₩5~50/회 | **10 Q-vic × 5회/일** | 50 | ₩1,500 |
| **마이크로 설문 (퀵폴)** | ₩3~10/건 | **10 Q-vic/건** | 무제한 | 변동 |
| **표준 설문 (5~10분)** | ₩300~1,000/건 | **500 Q-vic/건** | 무제한 | 변동 |
| **심층 설문 (30분+)** | ₩2,000~10,000/건 | **3,000 Q-vic/건** | 월 5건 한도 | ₩15,000 |
| **친구 추천 (가입)** | ₩2,000~10,000 | **1,500 Q-vic/명** | 무제한 | 무제한 |
| **친구 추천 (OKQ 전환)** | — | **3,000 Q-vic/명** | 무제한 | 무제한 |

**핵심 원칙:**
- **출석 + 검색 일일 상한 = ₩2,400** → 구독료 ₩1,500의 160%로 약간 여유 둠 (마이크로 활동만으로 흑자 가능)
- **설문·추천은 무제한** → 적극 사용자가 더 큰 가치 창출
- **시장 평균 대비 비슷한 수준 유지** → 사용자 기대치 충족, 어뷰징은 일일 상한으로 차단

## 1. 출석체크 · 만보기 시장 (앱테크)

### 1-1. 주요 서비스별 일일 적립 한도

| 서비스 | 일일 한도 | 주요 매커니즘 | 비고 |
|---|---|---|---|
| **토스 만보기** | **최대 100원** | 100보당 적립 + 구간별 랜덤 보상 (4천·6천·9천·1만보) | 7일 연속 출석 시 2,000보 구간 추가 |
| **캐시워크** | **최대 100캐시 (₩100)** | 100보당 1캐시, 1만보 한도 | + 광고 시청 5캐시+/회, 룰렛 40회 한도 |
| **캐시슬라이드** | 변동 | 잠금해제 시 자동 적립 + 광고 시청 보너스 | 정확한 단가 비공개 |
| **카카오뱅크 출석체크** | **~225원** | OX 퀴즈·데일리 용돈·광고시청 미션 통합 | 캠페인성, 상시 변동 |
| **돈이돼지·헬로페이 등** | ₩100~500 | 만보기+미션 결합 | 광고 의존도 높음 |

### 1-2. 통합 평균 — 실증 데이터

> **"전체 앱테크 이용자의 월 평균 수익은 약 ₩20,000"** (한국소비자원)
>
> **"전체 유저 하루 평균 약 ₩370"** — 가장 활발한 사용자는 ₩1,000+/일

**연령대별 차이:**
- 50대 평균 월 수익: ₩27,000 (1위)
- 60대 평균 월 수익: ₩24,000 (2위)
- 20~40대는 시간 대비 효율 낮아 활용도 ↓

**시사점:** 단순 출석·만보기는 **월 ₩600~3,000 수준이 시장 표준**. ₩30/일 출석 = 월 ₩900은 시장 중상위.

### 1-3. Q. 출석체크 설계 권장안

```
[Q. 출석체크 보상 구조]

기본 출석:        30 Q-vic/일  (월 ₩900)
연속 7일 보너스:  +100 Q-vic   (주간 보너스)
연속 30일 보너스: +500 Q-vic   (월간 그랜드)

→ 정상 출석 시 월 평균: ₩1,400~₩1,500
→ 구독료 ₩1,500 의 100% 안팎 상쇄 가능
```

**근거:** 토스(₩100/일·구간보너스 포함 시 월 ₩3,000+)·캐시워크(₩100/일 = 월 ₩3,000) 보다 약간 낮게 설계. 이유 — Q. 의 가치는 출석이 아니라 **명함 네트워크**에 있고, 출석은 락인 도구일 뿐. 과한 보상은 어뷰징 유인.

## 2. 검색 활동 보상 시장

### 2-1. 검색 보상 사례 — 제한적

검색 자체에 직접 보상을 주는 사례는 한국에서 드물지만, 유사 모델은 존재:

| 서비스 | 보상 방식 | 비고 |
|---|---|---|
| **네이버 페이 적립** | 검색 후 클릭 시 캐시백 (10~30원/회) | 쇼핑 검색만 |
| **다음 검색** | 캠페인성 적립 (변동) | 검색어 한정 |
| **버즈빌 광고 시청** | 광고 1편당 ₩50~200 | 검색 결과 페이지 광고 |

### 2-2. 검색 보상 설계의 어려움 — 어뷰징 위험

검색 보상은 가장 어뷰징되기 쉬운 활동:
- 봇이 무작위 키워드로 검색
- 클릭만 하고 즉시 이탈
- 동일 검색어 반복

→ **검증 매커니즘 필수:**
- 검색 결과 페이지 **체류 30초 이상**
- 검색 결과 **1회 이상 클릭**
- AI **검색 패턴 이상치 탐지** (동일 키워드 반복, 봇 시그니처)

### 2-3. Q. 검색 보상 설계 권장안

```
[Q. 검색 보상 구조]

조건 충족 검색:   10 Q-vic/회 (체류 30초 + 클릭 1회)
일일 한도:        5회 = 50 Q-vic/일 (월 ₩1,500)
누적 패턴 보너스: 1주일 매일 검색 시 +200 Q-vic
```

**근거:** 시장 단가(₩10~30/회) 와 동등 + 검증 강화. 50회/일 같은 과한 한도는 어뷰징 유인 → 5회로 제한.

## 3. 설문 패널 시장 (Q&A·데이터 수익화)

### 3-1. 주요 설문 패널 사례비 (실증)

| 서비스 | 단가 체계 | 비고 |
|---|---|---|
| **엠브레인 패널파워** | 탈락 ₩50 / 완료 변동 / 추첨 ₩5,000 | 국내 최대 패널 170만 명 |
| **패널나우** | 1회 최대 ₩10,000 / 랜덤참여 ₩700 / 퀵폴 ₩6 | 출금 ₩2,000부터 가능 |
| **헤이폴 (틸리언)** | 투표 1~100+P / 서베이 ~3,000P+ | SM C&C 운영 |
| **오픈서베이** | 변동 (B2B 위주) | 패널 대신 응답자에게 직접 지급 |

**월 참여 가능 횟수:** 평균 **20건/월** (패널나우 기준)

### 3-2. 단가 카테고리별 정리

```
[설문 단가 시장 매트릭스]

마이크로 설문 (1문항·10초):     ₩3~10/건       → 퀵폴·OX
표준 설문 (3~5문항·1~3분):       ₩100~700/건   → 일반 패널 설문
미디엄 설문 (10~15문항·5~10분):  ₩500~2,000/건 → 마케팅 조사
심층 설문 (30문항+·30분~):        ₩2,000~10,000/건 → 학술·정책 조사
좌담회 (대면 1~2시간):            ₩30,000~80,000/건 → 정성 조사 (FGI)
```

### 3-3. Q. 설문 설계 권장안

```
[Q. 설문 보상 구조]

마이크로 설문 (1일 1문):     10 Q-vic/건   (월 ₩300)
표준 설문 (주 2~3회):         500 Q-vic/건  (월 ₩5,000)
심층 설문 (월 1~2회 한정):    3,000 Q-vic/건 (월 ₩6,000)
좌담회 매칭 수익 배분:        ₩30,000+ (별도 정산)

→ 적극 사용자 월 설문 수익: ₩10,000~15,000
→ 구독료 ₩1,500 의 7~10배 흑자 가능
→ "설문이 진짜 돈이 된다" 메시지 성립
```

**근거:** 시장 평균(엠브레인·패널나우)과 유사 + Q. 데이터 자산의 차별성(명함 네트워크 + V-ID-Pro 전문가) 으로 광고주 CPL 상승 여지 → 마진 확보.

### 3-4. 설문 수익화 — Q. 만의 차별점

| 차원 | 기존 패널사 | **Q.** |
|---|---|---|
| 패널 검증 | 자체 가입 (가짜·중복 多) | **휴대폰 인증 + 명함 OCR + V-ID-Pro** |
| 타겟팅 정밀도 | 인구통계 기반 | **명함 OCR → 업종·직급·회사 정밀** |
| 광고주 신뢰도 | 평균 | **진성 직장인 패널 = 프리미엄** |
| **단가** | ₩100~2,000/건 | **₩500~5,000+ 가능 (프리미엄)** |

→ Q. 설문 패널은 **단가가 시장 평균 2~3배 받을 수 있는 잠재력** 보유.

## 4. 친구 추천 (Referral) 시장

### 4-1. 주요 서비스 친구 추천 보상

| 서비스 | 양쪽 보상 | 조건 |
|---|---|---|
| **쿠팡이츠** | 친구 ₩20,000 + 본인 ₩10,000 | 첫 주문 완료 시 |
| **토스 친구초대** | 양쪽 ₩5,000~10,000 (변동) | 신규 가입 + 19세 이상 |
| **토스 포스 (사장님용)** | 양쪽 ₩5,000 | 사업자 가입 |
| **카카오뱅크** | 변동 (캠페인성) | 통장 개설·체크카드 발급 |
| **케이뱅크** | ₩3,000~10,000 | 첫 입금 완료 |

### 4-2. 평균 친구 추천 보상 — ₩5,000 안팎

> 한국 친구 추천 시장의 **암묵적 표준은 양쪽 합산 ₩10,000~15,000** (각 ₩5,000~10,000)
>
> 이는 신규 사용자 CAC(고객획득비용) ₩30,000~50,000 의 1/3~1/2 수준에서 책정

### 4-3. Q. 친구 추천 설계 권장안

```
[Q. 친구 추천 보상 구조]

[1단계] 010Q메일 가입 시
  Scanner 보상: 1,500 Q-vic (= 월 1구독 가치)
  신규 가입자:   500 Q-vic (환영 보너스)

[2단계] OKQ메일 전환 시 (90일 내)
  Scanner 보상: 3,000 Q-vic (월 2구독 가치)
  신규 OKQ회원: 1,500 Q-vic (첫 달 무료 효과)

[3단계] 신규 회원이 1년 유지 시
  Scanner 보상: 5,000 Q-vic (1회 추가)

→ Scanner 1명 추천으로 최대 9,500 Q-vic 획득 가능
→ 시장 평균(₩5,000~10,000) 과 정합
→ 단, 마케팅사 별도 수수료(현금)와 분리
```

**근거:** 시장 평균에 맞추되 **단계별 분할 지급** 으로 어뷰징 방지 (가입 직후 이탈 시 1단계만 지급, 진성 사용자만 2~3단계).

## 5. Web3 토크노믹스 — 유통 vs 적립 균형 사례

### 5-1. 슈퍼워크 (Super Walk) — 가장 가까운 한국 사례

**핵심 구조:**
- 걷기 활동 → 토큰 적립
- 토큰 → LG U+ 통신비·교보생명 보험금 결제 (실사용)
- 락업/스테이킹 옵션 별도 제공

**시사점 (Q. 적용 가능):**
1. **실생활 결제처 확보** — 토큰이 "통신비처럼 쓰이는" 명확한 사용처
2. **B2B 제휴** — 적립 토큰을 외부 파트너가 흡수 (Q. 도메인·명함·커머스가 동일 역할)
3. **활동 → 보상 → 실사용 → 락인** 4단계 사이클

### 5-2. 유동성 스테이킹 (Liquid Staking) — DeFi 사례

**핵심 구조:**
- 토큰 락업 중에도 별도 자산(LST)으로 유동성 확보
- 락업 + 유통 양립

**Q. 적용:**
- Q-vic 잠긴 상태에서도 **Q-bit 미리 인출 가능 (할인율 적용)**
- 1년 락업 시 페널티 없음, 즉시 시 50% 페널티 (이미 제안)

### 5-3. 속도 방정식 시사점

리서치 결과 종합:
- **모든 성공한 한국 앱테크는 "월 ₩20,000 안팎"의 수익 한도** 를 유지
- 이 이상 주면 **광고주가 단가를 못 맞추고, 가짜 사용자가 늘어남**
- Q.는 시장 평균에 맞추되 **설문·추천에서 상한선 해제** 로 차별화

## 6. 종합 — Q-vic 보상 구조 최종 권장안

### 6-1. 일반 사용자 시나리오

| 사용자 유형 | 월 활동 | 월 Q-vic 적립 | 구독료 ₩1,500 상쇄 |
|---|---|---|---|
| **수동 사용자** | 출석만 | ₩900 | 60% 상쇄 — ₩600 현금 결제 |
| **보통 사용자** | 출석 + 검색 + 퀵폴 | ₩2,700 | 100% + 흑자 ₩1,200 |
| **활성 사용자** | + 표준 설문 5건 | ₩5,200 | 흑자 ₩3,700 |
| **적극 사용자** | + 추천 1명 + 심층설문 | ₩11,200 | 흑자 ₩9,700 |
| **파워 사용자** | 추천 10명 + 모든 활동 | ₩25,000~ | 흑자 ₩23,500+ |

**핵심:** 수동 사용자도 60% 자동 상쇄 — 구독 이탈률 낮춤. 활성+사용자는 흑자 — Q-bit 자산 전환 동기 자연 생성.

### 6-2. 어뷰징 방지 — 일일 상한 매트릭스

| 활동 | 일일 상한 | 검증 방식 |
|---|---|---|
| 출석 | 30 Q-vic (1회만) | 자정 1회 |
| 연속 보너스 | 100/500 Q-vic | 7일/30일 |
| 검색 | 50 Q-vic (10 × 5회) | 체류 30초 + 클릭 |
| 마이크로 설문 | 30 Q-vic (10 × 3건) | 1일 한도 |
| 표준 설문 | 무제한 | AI 응답 품질 검증 |
| 심층 설문 | 월 5건 한도 | 수동 큐레이션 |
| 친구 추천 | 무제한 | 단계별 분할 지급 + 1주 유지 검증 |

### 6-3. 매출 임팩트 시뮬레이션

```
[OKQ 100만 회원 시점 추정]

월 적립 Q-vic 총량:
  수동층 (30%): 30만 × ₩900   = ₩2.7억
  보통층 (40%): 40만 × ₩2,700 = ₩10.8억
  활성층 (20%): 20만 × ₩5,200 = ₩10.4억
  적극층 (8%):  8만  × ₩11,200= ₩9.0억
  파워층 (2%):  2만  × ₩25,000= ₩5.0억
  ────────────────────────────────────
  월 적립 총량: ₩37.9억

월 구독 매출:
  100만 × ₩1,500 = ₩15억

구독료 상쇄율 (Q-vic 사용):
  적립의 약 50% → ₩18.95억
  (이 중 80% = ₩15.16억이 구독 상쇄)

실제 현금 매출 (이니셜 구독):
  ₩15억 - ₩15.16억 ≈ 0?
```

**여기서 핵심 안전장치:** **이니셜 구독료는 Q-vic으로 100% 상쇄 불가** — 구독료의 **최소 50%는 현금** 필수.

→ **개정 권장:** 월 구독료 ₩1,500 중 **₩750(50%)은 반드시 현금**, 나머지 ₩750(50%)만 Q-vic 상쇄 허용.

이렇게 하면:
```
월 현금 매출 (100만 OKQ): 100만 × ₩750 = ₩7.5억
월 Q-vic 소진:           100만 × ₩750 = ₩7.5억 (회계 0 비용)
```

연간 **₩90억 안정적 현금 매출 + 활동 보상 시스템 유지**.

## 8. 시장 평균 정합성 검증

| 지표 | 한국 시장 평균 | Q. 설계 | 적합성 |
|---|---|---|---|
| 단순 출석 월 보상 | ₩600~3,000 | ₩900 | ✅ 시장 중하위 (어뷰징 방지) |
| 검색 월 보상 | ₩300~1,500 | ₩1,500 | ✅ 시장 상위 (차별화) |
| 설문 시간당 보상 | ₩2,000~6,000 | ₩3,000~6,000 | ✅ 시장 평균 |
| 친구 추천 1명 | ₩2,000~10,000 | ₩9,500 (3단계 합계) | ✅ 시장 상위 |
| 월 총 수익 (보통) | ₩20,000 | ₩2,700~5,200 | ⚠️ 시장 평균보다 낮음 |

**⚠️ 시장 대비 낮은 점이 의도된 이유:**
- 한국 앱테크는 **광고 수익에 의존** → 사용자가 광고에 노출되는 만큼 보상 (CPM·CPC)
- Q.는 **광고 수익이 주 모델 아님** → 명함·도메인·구독·설문 패널이 주 매출
- 시장 평균 수익을 줘버리면 광고 수익 없이 못 버팀
- 대신 **사용처(도메인·명함·커머스)** 를 매력적으로 만들어 **유보 동기** 부여

> **Q.의 전략:** "월 수익 ₩20,000" 보다 "월 ₩2,700이지만 1년 모으면 5자리 도메인 살 수 있음" 이 더 강력.
