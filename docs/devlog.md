## 2026-07-09

### 기간
- 2026-07-08 ~ 2026-07-09 연속 작업
- 대상: 010.Q 메인페이지, 요금제 카드, Founder 프리런칭 UX, 히어로 루프 타이틀

### 작업 배경
- 010.Q는 단순 명함서비스가 아니라 "번호가 메일이 되는 Q-mail 서비스"로 정의해야 한다.
- OK-Q와 Founder 플랜의 가입 조건, 혜택, 프리런칭 성격을 사용자가 한 화면에서 이해할 수 있게 정리할 필요가 있었다.
- 프리런칭 Founder Membership은 법인 설립 및 결제 연동 전 단계이므로, 화면 문구에서 "Open Stage"보다 "프리런칭"을 전면에 두는 방향으로 정리했다.

### 만든 것
- 010Q / OK-Q / Founder 요금제 카드 문구와 시각 계층 재정리
- 010Q 무료 플랜 조건 명시
  - 골드번호 소지자 우선
  - 일반번호도 Founder 추천 시 가입 가능
- OK-Q 플랜 정리
  - 가격을 `$1.00` 및 `(₩1,500고정)`으로 표시
  - `.00`은 작은 글꼴로 처리
  - `SMTP 이용 가능` 추가
  - `이니셜구독`을 FAQ 항목으로 연결
- Founder 카드 정리
  - `프리런칭 / 메인스테이지` 전환 버튼 추가
  - 프리런칭 파란 설명 박스 구성
  - 후원형 멤버십 신청금, 무료자동갱신, FM명예메일 표시
  - `20261996@Q.co.kr + WDH@Q.co.kr + 파운더 명예메일` 구조로 메일 혜택 표시
  - 파운더 명예메일 FAQ 연결
- Founder FAQ 확장
  - 프리런칭 기간, 회원자격 신청금, 기부금/투자금 아님, 정식 매출처리 전환 설명
  - FM001~FM999 명예메일 설명
- 히어로 루프 타이틀 개선
  - `골드번호와 골드도메인이 만나다`
  - `프리미엄 골드ID의 탄생`
  - `01010041004@Q.co.kr` 조립 애니메이션
  - 이어서 `10041004@Q.co.kr` 조립/깜빡임 애니메이션
  - `010, Q와 하나가 되다`
  - `내 번호가 메일이 되는 혁신, Q-메일`
- 컬러 포인트 조정
  - `골드번호` 중 `번호`는 블루
  - `골드ID` 중 `ID`는 블루
  - `2026` 오렌지, `1996` 골드
  - 조립형 번호 뒤쪽 `1004`는 골드

### 과정 메모
- 처음에는 Founder 혜택을 카드 안에 많이 담았으나, 카드가 과밀해져 "요약 카드 + FAQ 상세" 구조로 분리했다.
- `Open Stage`라는 내부 정책명은 방문자에게 직관성이 낮아, 화면에서는 `프리런칭`으로 통일했다.
- "후원금"이라는 표현은 오해 소지가 있어 `후원형 멤버십 신청금`으로 정리하고, FAQ에 기부금/투자금 아님을 고지했다.
- 배포는 Codex가 직접 하지 않고, 사용자가 `publish.bat`로 직접 수행하는 원칙을 유지했다.
