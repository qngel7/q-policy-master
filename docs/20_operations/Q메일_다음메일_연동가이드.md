# 📱 Q메일 다음메일(Hanmail) 앱 연동 가이드

안녕하세요! 발급받으신 나만의 평생 이메일, **Q메일(`내번호@q.co.kr`)**을 국민 메일 앱인 **다음(Daum) 메일 앱**에서 쉽게 확인하고 보낼 수 있도록 설정하는 방법입니다.

---

## 💡 준비물
1. **발급받은 Q메일 주소** (예: `01012345678@q.co.kr`)
2. **앱 전용 비밀번호** (발급 시 플랫폼에서 안내해 드린 알파벳/숫자 조합의 비밀번호)
   > ⚠️ **주의:** 평소 사용하시는 카카오/다음 로그인 비밀번호가 아닙니다. 반드시 **Q메일 앱 연동 전용으로 발급된 비밀번호**를 준비해 주세요.

---

## 🚀 연동 순서 (Step-by-Step)

### Step 1. 외부 계정 추가 진입
![다음메일 앱 계정 추가 이미지](./images/daum_step1.jpg)
1. 스마트폰에서 **다음 메일 앱**을 실행합니다.
2. 앱 좌측 상단의 **[가로선 3개(메뉴) 아이콘]**을 누른 뒤, 메뉴 좌측 하단의 **[⚙️ 환경설정]** 아이콘을 선택합니다.
3. 환경설정 메뉴에서 **[+ 메일 계정 추가]**를 누르고, 서비스 목록 맨 아래의 **[직접 입력]**을 선택합니다.

### Step 2. 계정 정보 입력 및 수동 설정 켜기
![이메일 주소 입력 및 수동 설정 이미지](./images/daum_step2.jpg)
1. 이메일 주소 칸에 **발급받으신 Q메일 주소 전체**(예: `010...`)를 입력합니다.
2. 비밀번호 칸에 발급받은 **앱 전용 비밀번호**를 붙여넣기 합니다.
3. 화면 하단에 있는 **[상세 설정(또는 수동 설정)]** 버튼이나 체크박스를 켭니다. *(이 과정을 거쳐야 서버 주소를 직접 넣을 수 있습니다!)*

### Step 3. 서버 정보 입력 (가장 중요 ⭐️)
상세 설정 화면이 열리면 아래 정보를 정확하게 입력해 주세요. (대소문자 및 띄어쓰기 주의)

![서버 정보 입력 화면 이미지](./images/daum_step3.jpg)

#### 📥 [받는 메일 서버 (IMAP)]
*   **서버 주소:** `imap.forwardemail.net`
*   **보안 연결(SSL) 사용:** `체크(활성화/ON)`
*   **포트:** `993`
*   **아이디:** 발급받은 Q메일 주소 전체 (`010...`)
*   **비밀번호:** 앱 전용 비밀번호

#### 📤 [보내는 메일 서버 (SMTP)]
*   **서버 주소:** `smtp.forwardemail.net`
*   **보안 연결(SSL) 사용:** `체크(활성화/ON)`
*   **포트:** `465`
*   **아이디:** 발급받은 Q메일 주소 전체 (위와 동일)
*   **비밀번호:** 앱 전용 비밀번호

### Step 4. 설정 완료 및 수/발신 테스트
![설정 완료 및 테스트 이미지](./images/daum_step4.jpg)
1. 모든 정보를 올바르게 입력하셨다면, 화면 상단 또는 하단의 **[저장(또는 완료)]** 버튼을 누릅니다.
2. 서버 확인이 통과되면 외부 메일함이 생성되며 동기화가 시작됩니다.
3. **메일 연동이 완료되었습니다!** 🎉 새 편지 쓰기를 통해 연동이 완벽하게 되었는지 본인의 다른 메일로 테스트해 보세요.

---

## ❓ 자주 묻는 질문 (Troubleshooting)
**Q. 보안 연결(SSL) 체크를 하니 포트 숫자가 마음대로 바뀝니다.**
> 다음 메일 앱 특성상 체크를 하면 기본 포트로 변경될 수 있습니다. 체크를 활성화하신 후 마지막에 포트 번호가 IMAP `993`, SMTP `465`로 되어 있는지 다시 한번 확인하시고 저장해 주세요.

**Q. 보낸 메일이 상대방의 스팸함으로 간다고 합니다.**
> Q메일은 글로벌 최고 수준의 스팸 방지 인증(DKIM, SPF, DMARC)이 적용되어 있습니다. 정상 세팅 시 상대방의 받은 편지함에 꽂히지만, 만약 지속적으로 스팸함에 들어간다면 플랫폼 관리자에게 문의하여 도메인 상태 점검을 요청해 주세요.
