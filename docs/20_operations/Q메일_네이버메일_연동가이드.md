# 📱 Q메일 네이버 메일 앱 연동 가이드

안녕하세요! 발급받으신 나만의 평생 이메일, **Q메일(`내번호@q.co.kr`)**을 한국에서 가장 많이 쓰이는 **네이버 메일 앱**에서 바로 확인하고 보낼 수 있도록 설정하는 방법입니다.

---

## 💡 준비물
1. **발급받은 Q메일 주소** (예: `01012345678@q.co.kr`)
2. **앱 전용 비밀번호** (발급 시 플랫폼에서 안내해 드린 알파벳/숫자 조합의 비밀번호)
   > ⚠️ **주의:** 평소 사용하시는 네이버 로그인 비밀번호가 아닙니다. 반드시 **Q메일 앱 연동 전용으로 발급된 비밀번호**를 준비해 주세요.

---

## 🚀 연동 순서 (Step-by-Step)

### Step 1. 외부 계정 추가 진입
![네이버 메일 앱 계정 추가 이미지](./images/naver_step1.jpg)
1. 스마트폰에서 **네이버 메일 앱**을 실행합니다.
2. 앱 좌측 상단의 **[가로선 3개(메뉴) 아이콘]**을 누릅니다.
3. 좌측 메뉴 상단(또는 하단)에 있는 **[⚙️ 톱니바퀴(환경설정)]** 아이콘을 선택합니다.
4. 설정 화면에서 **[메일 계정 추가]** 또는 **[다른 메일주소 추가]**를 선택한 후, 서비스 목록 맨 아래의 **[직접 입력(IMAP/POP3)]**을 누릅니다.

### Step 2. 수동 설정 및 서버 정보 입력 (가장 중요 ⭐️)
외부 메일 직접 입력 화면이 열리면 아래 정보를 꼼꼼하게 입력해 주세요. (대소문자 및 띄어쓰기 주의)

![이메일 주소 및 서버 정보 입력 화면 이미지](./images/naver_step2.jpg)

#### 📥 [받는 메일 서버 (IMAP)]
*   **서버 주소 (IMAP 서버):** `imap.forwardemail.net`
*   **포트:** `993`
*   **보안 연결 (SSL):** `체크(또는 사용함)`
*   **아이디 (사용자 이름):** 발급받은 Q메일 주소 전체 (`010...`)
*   **비밀번호:** 발급받은 **앱 전용 비밀번호**

#### 📤 [보내는 메일 서버 (SMTP)]
*   **서버 주소 (SMTP 서버):** `smtp.forwardemail.net`
*   **포트:** `465`
*   **보안 연결 (SSL):** `체크(또는 사용함)`
*   **아이디 (사용자 이름):** 발급받은 Q메일 주소 전체 (위와 동일)
*   **비밀번호:** 위와 동일한 **앱 전용 비밀번호**

### Step 3. 설정 완료 및 수/발신 테스트
![설정 완료 및 테스트 이미지](./images/naver_step3.jpg)
1. 모든 정보를 올바르게 입력하셨다면, 화면 상단 또는 하단의 **[확인(또는 완료)]** 버튼을 누릅니다.
2. 계정 확인이 정상적으로 통과되면, 네이버 메일 앱 좌측 메뉴에 Q메일 주소가 새 메일함으로 생성되고 동기화가 시작됩니다.
3. **메일 연동이 완료되었습니다!** 🎉 새 메일 쓰기를 열어 연동된 Q메일 주소로 다른 곳에 테스트 메일을 보내보세요.

## ❓ 자주 묻는 질문 (Troubleshooting)

**Q. 정보를 다 맞게 쳤는데 계정 인증에 실패했다고 뜹니다.**
> 비밀번호 앞뒤에 보이지 않는 띄어쓰기(공백)가 함께 복사되지 않았는지 한 번 지웠다가 다시 붙여넣기 해보세요. 또한 수신/발신 서버 양쪽 모두에 보안 연결(SSL) 옵션이 켜져 있는지 꼭 확인해야 합니다.

**Q. 네이버 메일 앱에서 보낸 메일의 보낸 사람 이름(표시 이름)을 바꿀 수 있나요?**
> 연동 후 네이버 메일 앱의 환경설정 > 해당 Q메일 계정 설정으로 들어가시면 '보내는 사람 이름'을 원하시는 이름이나 직함으로 자유롭게 변경하실 수 있습니다.
